FORD FIORASI – PROCESADOR DE ANTECEDENTES DISCIPLINARIOS
--------------------------------------------------------

USO ONLINE (STREAMLIT CLOUD):
1. Crear un repositorio en GitHub llamado "ford-fiorasi-antecedentes".
2. Subir estos archivos al repositorio.
3. Ingresar a https://streamlit.io/cloud y conectar tu cuenta de GitHub.
4. Crear una nueva app y elegir como archivo principal: app_fiorasi_web.py
5. Tu URL final será: https://ford-fiorasi-antecedentes.streamlit.app

USO LOCAL (opcional):
- Si tenés Python instalado, ejecutá:
  pip install -r requirements.txt
  streamlit run app_fiorasi_web.py
